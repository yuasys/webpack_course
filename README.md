# Udemy

「Webサイト制作のHTML / CSS / JSコーディングを一気に効率化する実践Webpack講座」の各セクションごとのブランチです。

- ブランチにcheckoutすることで、各セクションの作業状態を再現できます。
- セクションの概要を`README.md`に記載してますので、復習にお使いください。(下記リンクから閲覧できます)
